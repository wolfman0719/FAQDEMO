# FAQDEMO

FAQシステムのDocker Build デモサイト


## ビルドプロセス

### ビルド
```./build.sh```
### 実行
```./run.sh```

-bash: ./build.sh: Permission deniedとエラーになる場合は、

```chmod 777 *.sh```

を実行後、再度ビルドとランを実行
